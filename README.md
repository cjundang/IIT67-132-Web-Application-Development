# **1. ข้อมูลทั่วไปของรายวิชา**

## 1.1 คำอธิบายรายวิชา

ศึกษาแนวคิด หลักการ และเทคโนโลยีที่ใช้ในการพัฒนาโปรแกรมประยุกต์บนเว็บ ตั้งแต่โครงสร้างและมาตรฐานเว็บ สถาปัตยกรรมเว็บ การพัฒนาส่วนหน้า (Front-end) และส่วนหลัง (Back-end) การเชื่อมต่อฐานข้อมูล การสื่อสารข้อมูลระหว่างระบบ การรักษาความปลอดภัยของเว็บ การทดสอบและปรับปรุงประสิทธิภาพ รวมถึงการพัฒนาเว็บแอปพลิเคชันแบบสมบูรณ์ด้วยเครื่องมือและเฟรมเวิร์กมาตรฐาน ผ่านการปฏิบัติการและโครงงานจริง


# **2. จุดประสงค์และผลการเรียนรู้ของรายวิชา**

## 2.1 จุดประสงค์รายวิชา

เพื่อให้ผู้เรียนมีความรู้และทักษะในการออกแบบ พัฒนา ทดสอบ และประเมินโปรแกรมประยุกต์บนเว็บอย่างเป็นระบบ โดยคำนึงถึงสถาปัตยกรรมที่เหมาะสม มาตรฐานสากล ความปลอดภัย และความต้องการของผู้ใช้

## 2.2 ผลการเรียนรู้ของรายวิชา (CLOs)

เมื่อสิ้นสุดรายวิชา ผู้เรียนสามารถ:

* **CLO1:** อธิบายสถาปัตยกรรมเว็บ เทคโนโลยีเว็บ และมาตรฐานที่เกี่ยวข้อง
* **CLO2:** พัฒนาเว็บส่วนหน้าด้วย HTML, CSS, JavaScript และเฟรมเวิร์กที่เหมาะสม
* **CLO3:** พัฒนาเว็บส่วนหลังด้วยภาษาที่กำหนด พร้อมเชื่อมต่อฐานข้อมูลและบริการเว็บ
* **CLO4:** ประยุกต์เทคนิคด้านความมั่นคงปลอดภัยและการจัดการเซสชัน
* **CLO5:** ทดสอบ ประเมินประสิทธิภาพ และปรับปรุงเว็บแอปพลิเคชัน
* **CLO6:** ทำงานเป็นทีมและนำเสนอผลงานโครงงานเว็บได้อย่างมีประสิทธิภาพ


# **3. เนื้อหารายวิชาและการจัดการเรียนรู้**

## 3.1 หัวข้อเนื้อหาโดยสังเขป

1. ภาพรวมเทคโนโลยีเว็บและสถาปัตยกรรม Client–Server
2. โครงสร้าง HTML5 และการจัดการส่วนแสดงผลด้วย CSS
3. JavaScript เบื้องต้นและขั้นกลางสำหรับเว็บเชิงโต้ตอบ
4. Front-end Framework
5. Server-side Programming และสถาปัตยกรรม MVC
6. การเชื่อมต่อฐานข้อมูลและ ORM
7. การรักษาความปลอดภัยเว็บ
8. การทดสอบเว็บและการปรับปรุงประสิทธิภาพ
9. โครงงานพัฒนาเว็บแอปพลิเคชันแบบครบวงจร

## 3.2 วิธีการจัดการเรียนรู้

* การบรรยายและอภิปราย
* การสาธิตและปฏิบัติการในห้องปฏิบัติการ
* Problem-based Learning
* การทำงานกลุ่ม
* Project-based Learning
