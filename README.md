## **1. ข้อมูลทั่วไปของรายวิชา**

### 1.1 คำอธิบายรายวิชา

ศึกษาแนวคิด หลักการ และเทคโนโลยีที่ใช้ในการพัฒนาโปรแกรมประยุกต์บนเว็บ ตั้งแต่โครงสร้างและมาตรฐานเว็บ สถาปัตยกรรมเว็บ การพัฒนาส่วนหน้า (Front-end) และส่วนหลัง (Back-end) การเชื่อมต่อฐานข้อมูล การสื่อสารข้อมูลระหว่างระบบ การรักษาความปลอดภัยของเว็บ การทดสอบและปรับปรุงประสิทธิภาพ รวมถึงการพัฒนาเว็บแอปพลิเคชันแบบสมบูรณ์ด้วยเครื่องมือและเฟรมเวิร์กที่เป็นมาตรฐานสากล ผ่านการปฏิบัติการและโครงงานจริง

 

## **2. จุดประสงค์และผลการเรียนรู้ของรายวิชา**

### 2.1 จุดประสงค์รายวิชา

เพื่อให้ผู้เรียนมีความรู้และทักษะในการออกแบบ พัฒนา ทดสอบ และประเมินผลโปรแกรมประยุกต์บนเว็บอย่างเป็นระบบ โดยคำนึงถึงมาตรฐาน สถาปัตยกรรม ความปลอดภัย และความต้องการของผู้ใช้

### 2.2 ผลการเรียนรู้ (CLOs)

เมื่อสิ้นสุดรายวิชา ผู้เรียนสามารถ
**CLO1:** อธิบายสถาปัตยกรรมเว็บ เทคโนโลยีเว็บ และมาตรฐานที่เกี่ยวข้องได้อย่างถูกต้อง
**CLO2:** พัฒนาเว็บส่วนหน้าโดยใช้ HTML, CSS, JavaScript และเฟรมเวิร์กที่เหมาะสม
**CLO3:** พัฒนาเว็บส่วนหลังด้วยภาษาที่กำหนด พร้อมเชื่อมต่อฐานข้อมูลและบริการเว็บ
**CLO4:** ประยุกต์เทคนิคด้านความมั่นคงปลอดภัยและการจัดการเซสชันในเว็บแอปพลิเคชัน
**CLO5:** ทดสอบ ประเมินประสิทธิภาพ และปรับปรุงเว็บแอปพลิเคชัน
**CLO6:** ทำงานร่วมกันเป็นทีมและนำเสนอผลงานโครงงานเว็บได้อย่างเป็นระบบ

 

## **3. เนื้อหารายวิชาและกิจกรรมการเรียนรู้**

### 3.1 หัวข้อเนื้อหาโดยสรุป

1. ภาพรวมเทคโนโลยีเว็บและสถาปัตยกรรม Client–Server
2. โครงสร้างเว็บมาตรฐาน (HTML5) และรูปแบบการนำเสนอด้วย CSS
3. การพัฒนาเว็บเชิงโต้ตอบด้วย JavaScript
4. การจัดการข้อมูล การสื่อสาร และ RESTful API
5. การพัฒนาส่วนหลัง (Back-end) ด้วยภาษาและเฟรมเวิร์กที่สำนักวิชาใช้
6. การเชื่อมต่อฐานข้อมูลและ ORM
7. ความมั่นคงปลอดภัยเว็บ (Web Security Fundamentals)
8. การทดสอบเว็บ การวิเคราะห์ประสิทธิภาพ และการปรับปรุงระบบ
9. โครงงานพัฒนาเว็บแอปพลิเคชันแบบครบวงจร

### 3.2 วิธีการจัดการเรียนรู้

* การบรรยายเชิงทฤษฎี
* การสาธิตและปฏิบัติการในห้องปฏิบัติการ
* กิจกรรมแบบ Problem-based Learning
* การอภิปรายกลุ่มและการเรียนรู้แบบร่วมมือ
* โครงงานพัฒนาเว็บจริง (Project-based Learning)

## **6. การพัฒนารายวิชาและการประกันคุณภาพ**

* ปรับปรุงเนื้อหาให้ทันต่อเทคโนโลยีเว็บที่พัฒนาอย่างรวดเร็ว
